Goal:

Build a simple stock tracker that calculates the total investment based on manually defined stock prices.

Scope:

User inputs stock names and quantity.

Use a hardcoded dictionary to define stock prices (for example: { "AAPL": 180, "TSLA": 250 }).

Technology:

Python Programming Language,

Dictionary Data Structure,

Console Input and Output,

Basic Arithmetic Operations.
